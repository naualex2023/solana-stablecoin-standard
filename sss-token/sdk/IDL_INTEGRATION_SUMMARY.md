## Usage Example

```typescript
import { SSSTokenClient, AnchorProvider } from '@sss-token/sdk';

// Initialize SDK
const provider = AnchorProvider.env();
const sdk = new SSSTokenClient({ provider });

// Initialize stablecoin (fully typed)
const tx = await sdk.initialize(mint, authority, {
  name: "My Stablecoin",
  symbol: "MYST",
  uri: "https://example.com/metadata.json",
  decimals: 6,
  enablePermanentDelegate: true,
  enableTransferHook: true,
  defaultAccountFrozen: false
});

// All operations are now type-safe
const config = await sdk.getConfig(mint);
console.log(config.name, config.symbol);
```
